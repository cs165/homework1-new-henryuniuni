3.

Type of HTML elemnt
1. block ( p, h1, blockquote, ol, ul, table )

(1)Take full width of page

(2)Flow top to bottom

(3)Have height and width

(4)Allowed to have block or inline element

(5)Allowed to apply position ( CSS: float, position )

See the Pen ZPqwJV by henryuniuni (@HenryLin) on CodePen.


2. inline ( a, em, strong, br )

(1)Take up only as much width as needed

(2)Flows left to right

(3)Don't have height and width

(4)Not allowed to have block element

(5)Not allowed to apply position

See the Pen pYxGGG by henryuniuni (@HenryLin) on CodePen.


3. inline block ( img )

(1)Have height and width

(2)Allowed to have a block children

(3)Allowed to apply position

See the Pen VRERZb by henryuniuni (@HenryLin) on CodePen.
